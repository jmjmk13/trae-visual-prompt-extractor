# AI绘画提示词提取器 Skill

一个用于分析图片/视频、结构化文字想法、复盘生成结果并生成精准AI绘画提示词的 Agent Skill。内置基础风格词库，并融合 `vibeshotclub/vsc-skills` 的 620 条稀有亚种风格词，支持从构图、光影、材质、媒介、版式、空间和人物造型等维度反推出更准确的生图提示词，让小白也能拿到可直接复制使用的结果。

## ✨ 功能特性

- 🖼️ **图片分析**：精准识别风格、光影、构图、色调等
- 🎬 **视频分析**：自动抽取关键帧，逐帧分析
- 🧭 **无图构建**：根据文字想法补全主体、镜头、构图、光影和负面约束
- 🔍 **差距复盘**：对比目标与生成结果，找出跑偏原因并输出修正版提示词
- 🔒 **锚点锁定**：优先锁定视角、裁切、姿态、主体位置和视觉主次
- 🧬 **稀有风格匹配**：融合 VSC 风格词库，识别材质、媒介缺陷、亚文化造型、产品/空间/海报气质
- 📝 **多格式输出**：标准提示词、详细提示词、负面提示词、关键词、风格标签
- 📚 **海量词库**：基础风格词库 + 620 条 VSC 稀有风格词 + 新东方神怪水墨语法
- 🚀 **跨平台适配**：支持Midjourney、Stable Diffusion、Flux、DALL-E 3
- 🔍 **精确校验**：逐项确认视觉要素，确保提示词精准度

## 🚀 快速开始

### 方法1：手动安装

1. 将 `.trae/skills/visual-prompt-extractor/` 文件夹复制到你的 agent skills 目录

2. 重启对应 agent 或刷新技能列表

3. 在对话中上传图片或视频，AI会自动调用此skill进行分析

### 方法2：克隆仓库

```bash
git clone https://github.com/yourusername/trae-visual-prompt-extractor.git
cd trae-visual-prompt-extractor
cp -r .trae/skills/visual-prompt-extractor ~/.trae/skills/
```

## 📖 使用方法

### 基本用法

在 agent 对话中：

1. **上传图片**：直接拖入图片文件
2. **上传视频**：直接拖入视频文件
3. **输入指令**：
   - "分析这张图片的风格"
   - "提取这个视频的提示词"

## 🧭 三种工作模式

### 1. 图像拆解模式

当用户提供图片或视频时，优先提取构图、姿态、主体位置、视觉主次、风格和细节，避免只堆风格词导致"风格像但画面不像"。

### 2. 想法构建模式

当用户没有参考图时，将模糊想法补全为完整画面骨架：主体、动作、场景、镜头、构图、光影、色彩、材质、风格和负面约束。信息不足时会自动给出稳定默认值，并说明哪些内容是默认补全。

### 3. 差距复盘模式

当用户提供生成结果时，分析跑偏点和原因，指出需要前置锁定的锚点、需要弱化的词、需要加入的负面词，并输出下一版可直接复制的修正版提示词。

### 输出格式

分析完成后，会输出以下内容：

- 📝 **标准提示词**：逗号分隔的简洁提示词
- 📋 **详细描述**：完整的描述性提示词
- 🚫 **负面提示词**：排除低质量元素的提示词
- 🔒 **不可变锚点**：必须锁定的构图、姿态、主体位置等
- 🎚️ **可自由发挥区域**：允许模型发挥的配饰、纹样、氛围细节等
- 🏷️ **关键词**：核心关键词列表
- 🎭 **风格标签**：风格描述符
- ✅ **精确校验**：逐项确认的视觉要素
- 🖼️ **风格分析**：详细的风格分析报告
- 🎨 **艺术家参考**：匹配的艺术家参考
- 💡 **生成技巧**：生成类似作品的技巧
- 🚀 **Midjourney一键生成**：可直接复制使用的完整提示词

## 🎨 风格词库

词库包含以下类别：

### 艺术风格（200+）
- 经典流派：印象派、超现实主义、巴洛克、文艺复兴等
- 现代风格：赛博朋克、蒸汽朋克、合成波、蒸汽波等
- 地域文化：浮世绘、中国水墨、敦煌风格、国潮等
- 数字艺术：像素艺术、低多边形、故障艺术、分形艺术等

### 摄影风格（100+）
- 技术特征：胶片摄影、黑白摄影、HDR、长曝光等
- 色彩处理：复古色调、赛璐璐色调、双重曝光等
- 构图风格：黄金分割、三分法、极简构图等
- 场景类型：肖像、风景、街头、建筑等

### 光影类型（50+）
- 自然光：黄金时刻、蓝调时刻、月光、极光等
- 人工光：霓虹灯、钨丝灯、聚光灯、环形灯等
- 光影效果：逆光、伦勃朗光、丁达尔效应等

### 艺术家参考（50+）
- 经典艺术家：莫奈、梵高、达利、毕加索等
- 现代艺术家：新海诚、宫崎骏、大友克洋等
- 数字艺术家：Beeple、WLOP、Guweiz等
- 摄影大师：安塞尔·亚当斯、安妮·莱博维茨等

### VSC稀有风格词（620）
- 材质与表面质感：液态铬金属、螺钿镶嵌、烟熏玻璃、树脂封存标本等
- 影像与摄影缺陷：湿版火棉胶、过期宝丽来、VHS磁带故障、CRT荧光屏等
- 动画/漫画/插画亚种：用于识别比“二次元/动漫”更细的画风来源
- 平面、产品、空间、时装与亚文化：用于反推出包装、海报、橱窗、妆造、场景气质
- 新东方神怪水墨语法：用于山海经、神怪、异兽、工笔水墨、矿物颜料和留白烟云类画面

## 🌐 跨平台提示词适配

### Midjourney
```
提示词 --ar 3:4 --v 6 --q 2 --style raw
```

### Stable Diffusion
- 使用逗号分隔格式
- 同时包含正面和负面提示词
- 使用权重如 `(关键词:1.5)`

### Flux
- 使用自然语言描述
- 较少依赖逗号分隔标签

### DALL-E 3
- 使用自然语言描述
- 更具描述性和叙事性
