# AI绘画创作手法分析器
## 核心定位
不是"复刻提示词工具"，而是"创作手法学习工具"。
通过分析一张好图，理解它好在哪、用了什么手法、我能学到什么。

## 核心方法：六维细节观察法
不分类、不贴标签，直接看细节。每个维度都必须找到**具体的视觉证据**。

### 0. 风格/审美观察（先看整体感觉）
必须回答：
- 整体给人的感觉是什么？（清冷/温暖/华丽/简约/国风/日系/欧美等）
- 有没有明显的文化/地域风格？（中国风/和风/欧美写实等）
- 人物造型有什么特点？（脸型/五官/发型/服饰）
- 构图和取景有什么特点？（特写/全景/对称/留白）
- 整体氛围是什么？（梦幻/写实/浪漫/暗黑等）

### 1. 边缘观察
必须回答：
- 整体边缘是偏硬还是偏柔？
- 哪里边缘硬？哪里边缘柔？为什么？
- 轮廓线是清晰的还是融合在背景里的？
- 有没有"硬边切割"的地方？在哪里？

### 2. 色彩观察
必须回答：
- 最亮的地方是什么颜色？（不是"白色"，是"暖白/冷白/粉白"）
- 最暗的地方是什么颜色？（不是"黑色"，是"深蓝黑/紫黑/棕黑"）
- 中间调是什么颜色倾向？
- 阴影是什么颜色？（不是"灰色"，是"紫灰/蓝灰/棕灰"）
- 高光是什么颜色？（不是"白色"，是"暖黄白/冷青白/粉白"）
- 主色有哪3-5种？大概比例？

### 3. 纹理观察
必须回答：
- 哪里有颗粒感/笔触纹理？哪里是光滑的？
- 纹理是粗的还是细的？均匀还是局部？
- 皮肤是哑光的还是有光泽的？通透吗？
- 头发是一绺一绺的还是有笔触感的？
- 布料/金属/宝石分别是什么质感？

### 4. 光感观察
必须回答：
- 光从哪个方向来？（左上/右上/正前/逆光）
- 最亮的点在哪里？
- 阴影是虚的还是实的？软边还是硬边？
- 有没有体积感？怎么体现的？
- 有没有透光/反光/折射？在哪里？

### 5. 焦点观察
必须回答：
- 视觉焦点是什么？（第一眼看到哪）
- 哪里最清晰？哪里最模糊？
- 用了什么手法突出焦点？（对比/色彩/细节/位置）
- 哪里是"虚"的？为什么要虚？
- 画面的"呼吸感"在哪里？

## 使用原则
1. **用眼睛看，不用脑子分类**——看到什么说什么，不往"厚涂/赛璐璐"这类标签里套
2. **每条结论都要有视觉证据**——不能说"质感好"，要说"头发上有颗粒状笔触纹理"
3. **重点学"为什么这么做"**——而不是"用了什么词"
4. **练习提示词是用来练手的**——能学到手法就行，不追求1:1还原
5. **诚实面对没看懂的地方**——不确定就说"不确定"，不要瞎编

## 样图风格特征参考（从D:\web3\projects\样图分析得出）
以下是样图中反复出现的风格特征，可以作为分析时的参考：

### 艺术风格
- **基础风格**：日系动漫风格 + 数字绘画技巧融合
- **常见元素**：哥特洛丽塔服饰、日式和服、未来科技元素
- **构图特点**：近景/特写肖像，前景常有道具（花、剑、星球、扇子）

### 色彩特征
- **背景**：深色/纯黑背景为主，突出主体
- **主色**：蓝色、紫色、黑色为主，搭配彩虹色/金色点缀
- **对比**：高对比度，暗色背景衬托亮色主体
- **眼睛**：彩虹色虹膜（红橙黄绿蓝紫渐变），宝石般的质感

### 光感特征
- **光源**：侧光/逆光为主，创造轮廓光
- **特殊光效**：镜头光晕、发光粒子、彩虹色散、镜面反射
- **皮肤光感**：柔和高光，阴影带色彩倾向（紫灰/蓝灰）

### 纹理特征
- **皮肤**：细腻光感，通透但不油光
- **头发**：丝质光泽，有笔触纹理但光滑
- **材质**：金属（哑光金/暗金）、宝石（通透内部折射）、布料（褶皱清晰）

### 视觉焦点
- **眼睛**：绝对焦点，细节最丰富，色彩最鲜艳
- **前景道具**：第二焦点，常带有特殊效果（反射、发光）
- **虚实对比**：焦点清晰，背景模糊，前景道具半虚半实

## 重要提醒
- 分析的准确性比全面性重要
- 学到3个真正有用的技巧，比列出20个模糊标签强
- 如果某张图某个维度没什么特别的，可以简单带过
- 重点关注"这张图和普通图不一样的地方"
