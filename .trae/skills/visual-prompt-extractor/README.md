# Visual Prompt Extractor Skill

## 简介

这是一个用于精准分析图片/视频、结构化文字想法、复盘生成结果并生成AI绘画提示词的 Agent Skill。它在基础风格词库之外内置了稀有亚种风格词，用来识别材质、媒介缺陷、亚文化造型、版式、空间和特殊影像气质，并帮助小白拿到可直接复制使用的提示词。

## 功能特性

- ✅ 图片分析：精准识别风格、光影、构图、色调等
- ✅ 视频分析：自动抽取关键帧，逐帧分析
- ✅ 无图构建：把文字想法补成完整画面骨架
- ✅ 差距复盘：分析生成结果跑偏原因并输出修正版提示词
- ✅ 锚点锁定：优先锁定视角、裁切、姿态、主体位置和视觉主次
- ✅ 稀有风格匹配：用620条亚种风格词补强“说不上来但很关键”的风格质地
- ✅ 多格式输出：标准提示词、详细提示词、负面提示词、关键词、风格标签
- ✅ 海量词库：基础风格词库 + 稀有亚种风格词库 + 新东方神怪水墨语法
- ✅ 跨平台适配：支持Midjourney、Stable Diffusion、Flux、DALL-E 3

## 如何使用

### 在agent中使用

1. 将此skill文件夹复制到你的 agent skills 目录：
   ```
   .trae/skills/visual-prompt-extractor/
   ```

2. 在对话中上传图片或视频，AI会自动调用此skill进行分析

3. 可选指令示例：
   - "分析这张图片的风格"
   - "提取这个视频的提示词"

## 工作模式

### 图像拆解模式

用户提供图片/视频时，先锁定构图、姿态、主体位置、视觉主次，再匹配风格、光影、色彩和材质，避免只提取风格导致相似度不足。

### 想法构建模式

用户没有参考图时，先搭建画面骨架：主体、动作、场景、镜头、构图、光影、色彩、材质、风格和负面约束。信息不足时自动补合理默认值，并说明默认了什么。

### 差距复盘模式

用户提供生成结果时，先分析跑偏点，再给出下一版需要锁定的锚点、需要弱化的词、需要新增的负面提示词，并输出可直接复制的修正版。

### 给别人使用

#### 方法1：分享整个skill文件夹

将 `.trae/skills/visual-prompt-extractor/` 整个文件夹打包发送给其他人，他们只需：

1. 解压到自己的 `.trae/skills/` 目录
2. 重启对应 agent 或刷新技能列表
3. 即可使用

#### 方法2：上传到GitHub

1. 创建GitHub仓库
2. 上传skill文件夹内容
3. 其他用户克隆到本地 `.trae/skills/` 目录即可使用

#### 方法3：作为公开skill提交

提交到目标 agent 或插件市场的官方审核渠道，审核通过后可成为公开可安装的skill。

## 词库内容

### 艺术风格（200+）
- 经典流派：印象派、超现实主义、巴洛克、文艺复兴等
- 现代风格：赛博朋克、蒸汽朋克、合成波、蒸汽波等
- 地域文化：浮世绘、中国水墨、敦煌风格、国潮等
- 数字艺术：像素艺术、低多边形、故障艺术、分形艺术等

### 摄影风格（100+）
- 技术特征：胶片摄影、黑白摄影、HDR、长曝光等
- 色彩处理：复古色调、赛璐璐色调、双重曝光等
- 构图风格：黄金分割、三分法、极简构图等
- 场景类型：肖像、风景、街头、建筑等

### 光影类型（50+）
- 自然光：黄金时刻、蓝调时刻、月光、极光等
- 人工光：霓虹灯、钨丝灯、聚光灯、环形灯等
- 光影效果：逆光、伦勃朗光、丁达尔效应等

### 艺术家参考（50+）
- 经典艺术家：莫奈、梵高、达利、毕加索等
- 现代艺术家：新海诚、宫崎骏、大友克洋等
- 数字艺术家：Beeple、WLOP、Guweiz等
- 摄影大师：安塞尔·亚当斯、安妮·莱博维茨等

### 稀有亚种风格词（620）
- 材质与表面质感：透明、半透明、金属、陶瓷、树脂、织物、锈蚀、虹彩等
- 摄影工艺与影像缺陷：胶片、宝丽来、CRT、VHS、半调印刷、热成像、X光等
- 动画、漫画、插画、平面、产品、空间和亚文化造型等更细分的风格标签
- 山海经、神怪、异兽、工笔水墨、矿物颜料、留白烟云可参考 `shan-ze-style-grammar.md`
