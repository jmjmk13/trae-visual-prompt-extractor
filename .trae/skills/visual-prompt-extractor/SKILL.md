# AI绘画提示词提取器

本skill用于分析上传的图片或视频，也用于把用户的文字想法结构化成可生成的AI绘画提示词，并能复盘生成结果与目标之间的差距。通过内置的海量风格词库和画面锚点流程，确保提示词既专业、精准，又适合小白直接复制使用。

## 何时调用

- 用户上传图片或视频文件
- 用户要求"分析这张图片的风格"
- 用户要求"提取这张图片的提示词"
- 用户要求"生成这张图片的AI绘画提示词"
- 用户提供图片URL并请求提示词分析
- 用户没有参考图，只描述想要的画面、角色、风格或氛围
- 用户要求"帮我写一个直接复制的提示词"
- 用户提供生成结果并要求"差距在哪"、"复盘一下"、"改到更像目标"

## 工作流程

### 1. 接收输入

**图片文件**:
- 支持格式: JPG, JPEG, PNG, GIF, BMP, WEBP
- 使用视觉能力读取图片内容

**视频文件**:
- 支持格式: MP4, MOV, AVI, MKV, WEBM
- 提取关键帧（每5秒提取一帧，最多10帧）
- 对每个关键帧单独分析

**文字想法**:
- 用户可能只提供模糊需求，例如"想要一个浮世绘白发少女"
- 不要因为信息不完整而频繁追问；优先补合理默认值并明确标注
- 只有当缺失信息会显著改变结果时，最多先问一个关键问题

**生成结果复盘**:
- 用户可能同时提供目标图、生成图、提示词或只提供生成图
- 对比时先找"为什么不像"，再重写可复制提示词

### 2. 判断工作模式

先判断任务属于哪种模式，然后按对应优先级生成结果：

| 模式 | 触发场景 | 目标 | 优先级 |
|------|----------|------|--------|
| 图像拆解模式 | 有参考图/视频，要求提取或复刻提示词 | 准确还原图像核心视觉 | 构图 > 姿态 > 主体位置 > 视觉主次 > 风格 > 细节 |
| 想法构建模式 | 没有参考图，只有文字想法 | 把模糊想法补成完整画面规格 | 主体 > 动作 > 场景 > 镜头 > 构图 > 光影 > 风格 > 负面约束 |
| 差距复盘模式 | 有生成结果，要求分析差距或重写 | 找出跑偏原因并给修正版 | 目标差异 > 跑偏原因 > 锁定锚点 > 负面约束 > 修正版提示词 |

### 3. 画面骨架与锚点

在生成提示词之前，必须先建立画面骨架。画面骨架用于防止提示词只堆风格词、却丢失用户真正想要的画面。

**画面骨架字段**:
- 主体：人物/动物/物体/场景是什么，数量是多少
- 动作/状态：姿态、表情、手部动作、身体方向、运动趋势
- 场景：背景、环境、时代、空间关系
- 镜头：远景/全身/半身/中近景/特写，俯视/仰视/平视，广角/鱼眼/长焦
- 构图：中心/偏置/对角线/旋转/三分法/黄金分割/紧裁切
- 光影：光源方向、软硬、对比度、高光与阴影位置
- 色彩：主色、辅色、点缀色、饱和度、冷暖关系
- 材质：丝绸、金属、玻璃、皮革、毛发、水面等
- 风格：艺术流派、文化风格、数字风格、艺术家参考
- 质量与平台：Midjourney、Stable Diffusion、Flux、DALL-E 3等

**不可变锚点**:
- 图像拆解模式：提取6个不可变锚点，包括视角、裁切、姿态、脸部/主体位置、手部/关键动作、主体与背景主次
- 想法构建模式：从用户文字中识别最重要的3-5个意图锚点，不足时补默认值
- 差距复盘模式：把目标与生成结果的差异转成下一版必须锁定的锚点
- 不允许风格词覆盖锚点；复刻类任务中，姿态和构图永远比"华丽、精致、超详细"更重要

**可自由发挥区域**:
- 明确哪些元素可以让模型自由发挥，例如小配饰、背景纹样、局部花纹、气氛细节
- 对容易跑偏的元素加边界词，例如 limited, controlled, subtle, not overcrowded

### 4. 逐项精确校验

**在生成提示词之前，必须逐项确认以下要素的准确性：**

| 校验项 | 确认要点 | 示例 |
|--------|----------|------|
| 光源方向 | 确认光源来自哪个方向（左上/右上/正上/侧光/逆光） | ✅ 光源来自左上，在头发和脸颊形成高光 |
| 光影类型 | 确认是硬光还是柔光，阴影是否柔和 | ✅ 柔和高光，微妙阴影（非硬阴影） |
| 主体颜色 | 精确描述主体颜色，避免笼统描述 | ✅ 纯白色头发（非灰色），左眼紫色带粉色反光 |
| 材质质感 | 确认材质类型（丝绸/金属/玻璃/布料等） | ✅ 黑色丝绸和服，镜面反射效果的刀 |
| 反射效果 | 确认是否有反射/折射效果 | ✅ 刀身清晰反射花朵和头发 |
| 构图方式 | 确认构图类型和视角 | ✅ 特写肖像，刀水平横跨下半脸 |
| 风格类型 | 精确匹配风格词库中的风格 | ✅ 写实数字绘画风格，结合动漫人物设计 |

**注意**: 如果发现任何要素描述不准确，必须重新观察图片，直到确认所有要素精准无误。

### 5. 分析视觉要素

系统分析以下方面：

**主体分析**:
- 主体内容（人物、动物、物体、风景）
- 主体特征（姿势、表情、服装、配饰）
- 主体数量及相互关系

**风格分析**（匹配style_library.md）:
- 艺术风格（印象派、超现实主义、动漫风格、数字绘画等）
- 摄影风格（肖像、风景、街头、电影感等）
- 文化风格（浮世绘、中国水墨、敦煌风格、国潮等）
- 数字风格（像素艺术、低多边形、故障艺术等）

**光影分析**:
- 自然光（黄金时刻、蓝调时刻、月光、极光等）
- 人工光（霓虹灯、钨丝灯、聚光灯、环形灯等）
- 光线方向（逆光、侧光、顶光等）
- 光影效果（剪影、镜头光晕、丁达尔效应等）

**色彩分析**:
- 色彩调色板（暖色/冷色、高饱和/低饱和）
- 具体色调（匹配style_library.md中的色彩参考）
- 对比度（高/低/中）

**构图分析**:
- 取景方式（黄金分割、三分法、居中构图等）
- 视角（正面、侧面、鸟瞰、仰视）
- 景深（浅景深虚化、深景深清晰）

**材质分析**:
- 表面纹理（光滑、粗糙、光泽、哑光）
- 材质类型（丝绸、皮革、金属、玻璃、毛发等）

**氛围分析**:
- 情绪基调（宁静、戏剧性、忧郁、充满活力）
- 环境效果（雾气、雨水、尘埃、闪光）

**艺术家参考分析**:
- 将风格匹配到知名艺术家（参考style_library.md）
- 使用"风格类似"或"受...风格影响"来引用艺术家

### 6. 小白友好补全规则

当用户没有参考图、描述不完整或不知道怎么表达时，不要直接输出一堆抽象风格词。先自动补全一版稳定默认方案，并告诉用户哪些地方是默认补全。

**默认补全建议**:
- 默认镜头：中近景或半身构图，主体清晰
- 默认光影：柔和定向光，避免强硬阴影破坏主体
- 默认构图：主体明确、背景不抢戏，必要时使用紧裁切
- 默认细节密度：主体高细节，背景中等细节
- 默认负面约束：低质量、畸形手、主体不清、过度装饰、风格混乱
- 默认平台：用户未指定时，优先给"直接复制通用版"，再给Midjourney版

**少问原则**:
- 信息缺失但不影响主要方向时，直接补默认值
- 信息缺失会导致完全不同画面时，只问一个关键问题
- 问题之后也要给一个"如果你不想选，我建议用..."的默认选项

### 7. 生成提示词

生成多种格式的提示词：

#### 格式A：直接复制提示词

结构: `[不可变锚点/画面骨架] + [主体] + [动作/状态] + [镜头/构图] + [风格] + [光影] + [色彩] + [细节边界] + [质量标签]`

复刻或相似度优先时，把构图、姿态、裁切、脸部位置、手部动作放在最前面。

示例:
```
动漫女孩，纯白色光泽长发编成粗辫子，异色瞳（左眼紫罗兰紫色带粉色反光，右眼祖母绿带金色反光），手持日本武士刀水平横过下半脸，黑色丝绸和服配金色花卉刺绣，鲜红色彼岸花装饰，左上方向光源，柔和高光，微妙阴影，刀身镜面反射花朵和头发，深色神秘背景，写实数字绘画风格，杰作，超详细，8K
```

#### 格式B：详细描述提示词

更全面的版本，包含更多上下文：

示例:
```
一幅精美的特写肖像，描绘一位拥有纯白色光泽长发的动漫女孩，头发编成一条优雅的粗辫子垂在身侧。她的眼睛是迷人的异色瞳——左眼是生动的紫罗兰紫色，带有柔和的粉色反光；右眼是明亮的祖母绿色，带有金色闪烁。她手持一把传统的日本武士刀，水平横过脸部下方，刀身如镜面般光滑，清晰地反射出周围的红色彼岸花和几缕发丝。刀柄为黑色，配有华丽的金色配件。她穿着一件奢华的黑色丝绸和服，上面装饰着精致的金色花卉刺绣图案。鲜红色的彼岸花插在她的头发中，并散落在深色背景中。光线从左上方照射下来，在她的头发和左脸颊上形成柔和的高光，同时在脸部右侧形成柔和、扩散的阴影。背景深色调，烘托主体。采用超详细的数字绘画风格，具有逼真的质感和光影效果。
```

#### 格式C：负面提示词

排除低质量元素的提示词：

示例:
```
低质量，最差质量，糟糕的人体结构，比例失调，多余肢体，缺失肢体，融合肢体，漂浮肢体，断开的肢体，畸形的手，多余手指，缺少手指，额外数字，更少数字，模糊，文字，水印，签名，logo，裁剪不当，超出画面，草稿，草图，未完成，低分辨率，JPEG伪影，用户名，错误，变异，变形，丑陋，毁容，死亡，恐怖，血腥，暴力，照片写实，3D渲染，CG，糟糕的艺术，业余，平面色彩，饱和度低，灰色头发，暗淡的眼睛，无反射，哑光刀身，硬阴影，过曝，欠曝，卡通风格，Q版
```

#### 格式D：关键词列表

提取10-20个核心关键词：

示例:
```
纯白色光泽长发，粗辫子，异色瞳，紫色眼睛，绿色眼睛，日本武士刀，黑色丝绸和服，金色花卉刺绣，红色彼岸花，左上光源，柔和高光，微妙阴影，镜面反射，深色神秘背景，数字绘画，写实渲染
```

#### 格式E：风格标签

提取5-10个风格描述符：

示例:
```
数字绘画，写实动漫风格，柔和定向光源，反射表面，日本美学，深色神秘氛围，超详细质感
```

#### 格式F：小白解释版

用简单语言说明这条提示词为什么这样写，提醒用户哪些词负责主体、哪些词负责构图、哪些词负责防跑偏。

#### 格式G：差距修正版

当用户提供生成结果复盘时，必须输出：
- 跑偏点：逐条说明哪里不像
- 原因：指出是提示词缺失、权重过强、负面约束不足，还是模型默认审美导致
- 修正策略：哪些词提前、哪些词删减、哪些词加入负面
- 可直接复制的下一版提示词

### 8. 参考风格词库

分析时必须查阅 `style_library.md` 获取精确术语：

**艺术风格匹配**:
- 如果图片显示柔和笔触和光影强调 → "印象派，莫奈风格"
- 如果图片有梦幻、奇异的意象 → "超现实主义，达利风格"
- 如果图片是动漫风格带平涂 → "动漫风格，赛璐璐着色"
- 如果图片有霓虹灯和未来城市 → "赛博朋克"

**摄影风格匹配**:
- 如果图片有胶片颗粒和复古色彩 → "胶片摄影，柯达色调"
- 如果图片是黑白高对比度 → "黑白摄影"
- 如果图片有电影感灯光和构图 → "电影感摄影"

**光影匹配**:
- 如果是温暖柔和的日落/日出光芒 → "黄金时刻光线"
- 如果是暮光冷色调 → "蓝调时刻光线"
- 如果是逆光主体带光晕 → "逆光，剪影"

**艺术家匹配**:
- 动漫风格带详细背景和情感天空 → "新海诚风格"
- 奇幻风格带魔法世界 → "宫崎骏风格"
- 未来感镀铬机器人 → "空山基风格"

## 质量指南

1. **精确性**: 使用style_library.md中的精确术语，避免模糊描述
2. **完整性**: 覆盖所有视觉要素（主体、风格、光影、色彩、构图）
3. **专业性**: 使用行业标准术语
4. **可操作性**: 确保提示词可直接用于AI绘画工具
5. **特异性**: 尽可能识别并包含具体的艺术家、风格、技术
6. **相似度优先**: 复刻类任务中，构图、姿态、裁切和主体位置优先于风格词
7. **小白可用**: 输出必须包含直接复制版，并说明默认补全与防跑偏词

## 跨平台提示词适配

### Midjourney
- 添加 "--ar X:Y" 设置宽高比
- 添加 "--v 6" 或 "--v 5" 指定版本
- 添加 "--q 2" 提高质量
- 使用简洁的逗号分隔格式
- 可以使用 "by [艺术家名]" 格式

### Stable Diffusion
- 使用逗号分隔格式
- 需要时使用 "BREAK" 分隔段落
- 同时包含正面和负面提示词
- 使用权重如 "(关键词:1.5)" 来强调

### Flux
- 更自然语言友好
- 可以使用描述性句子
- 较少依赖逗号分隔标签

### DALL-E 3
- 使用自然语言描述
- 更具描述性和叙事性
- 避免过多技术术语
- 注重视觉描述而非风格标签

## 重要注意事项

1. **必须查阅style_library.md** - 在生成提示词之前确保术语准确
2. **风格精准匹配** - 存在具体术语时不要使用通用术语
3. **包含质量标签** - 添加"杰作，最佳质量，超详细"等提升结果质量
